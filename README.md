# Medicine Management

   Maintain medicine details in a shop. Allows you to add/remove/update/get any medicine in the shop

## Features
* Add a medicine
* Search a medicine
* Update attributes of a medicine
* Remove a medicine


## Packages Used
* Express
* Mongoose
* SuperTest
* Swagger-ui-express
* Istanbul
* Http-status-code
* Sinon
* Should
* nyc

API documentation could be reached at http://localhost:3000/api-docs/
